# Diwali Sales Data Analysis

This project performs Exploratory Data Analysis (EDA) on a Diwali Sales dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Female customers spend more than male customers
- Age group 26–35 has the highest purchasing power
- Top states contributing to sales
- Top product_category from sales
- Occupation on which amount spend more

  - Gender vs Spending
- Age Group vs Orders
- State vs Sales
- Occupation vs Amount
